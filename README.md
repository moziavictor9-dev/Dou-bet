# DOU Bet

DOU Bet is a simple sports betting platform designed for students of Dennis Osadebay University.

## Overview
This project is a prototype betting platform where students can view matches and place bets.

## Features
- View available football matches
- See betting odds
- Place bets
- Simple user interface

## Tech Stack
- HTML
- CSS
- JavaScript
